# Stakeholder Register Template

## Purpose
Maintains a structured directory of all project stakeholders, analyzing their influence, interest, and communication needs.

## Traceability
Traces back to: Project Profile. Traces forward to: Elicitation Plan, Meeting Minutes.

## 2. Power/Interest Grid Analysis
Categorize stakeholders into four quadrants: Manage Closely, Keep Satisfied, Keep Informed, or Monitor only.

## 3. Communication Strategy
Define how and when each stakeholder group will be engaged (e.g. weekly reports, steering calls, slack).

## Optional Sections
## 4. Impact Mapping
Outline how specific stakeholder processes will change after solution implementation.

- [ ] Sponsors and key decision-makers are identified.
